# Portfolio Frontend

A personal portfolio website built with React and Vite, featuring a public-facing portfolio and an admin dashboard.

## Tech Stack

- **React 18** + **Vite**
- **React Router v6** — client-side routing
- **Ant Design** — UI components
- **Framer Motion** — animations
- **Recharts** — analytics charts
- **Axios** — API requests

## Features

- Public portfolio pages: About, Projects, Experience, Education, Certificates
- Admin dashboard to manage all content (CRUD)
- Visit analytics chart
- JWT-based admin authentication

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:5000
```

## Deployment

Hosted on **Vercel**. Set `VITE_API_URL`

```bash
npm run build
```
